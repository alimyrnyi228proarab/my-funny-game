# my-funny-game


Простая консольная игра на Python, в которой нужно угадать случайное число.

📌 Описание

Программа загадывает число от 1 до 100, а у игрока есть 7 попыток, чтобы его угадать.
После каждой попытки выводится подсказка — больше или меньше загаданного числа.

🚀 Запуск
Установи Python 3

Сохрани код в файл:

guess_number.py

Запусти:

python guess_number.py
🧠 Как работает код

Разберём программу по частям:

1. Импорт модуля
import random

Подключаем встроенный модуль random, который нужен для генерации случайного числа.

2. Генерация числа и количество попыток
number = random.randint(1, 100)
attempts = 7
random.randint(1, 100) — создаёт случайное число от 1 до 100 (включительно)
attempts = 7 — задаём количество попыток
3. Приветственное сообщение
print("Угадай число от 1 до 100. У тебя 7 попыток.")

Просто вывод текста для игрока.

4. Основной цикл игры
while attempts > 0:

Цикл работает, пока у игрока есть попытки.

5. Ввод числа
guess = int(input("Введи число: "))
input() — получает ввод пользователя
int() — преобразует строку в число

Если ввести не число, программа выдаст ошибку (это можно улучшить позже).

6. Проверка ответа
if guess == number:
    print("Ты угадал!")
    break
Если число угадано — выводим сообщение
break — завершает цикл
7. Подсказки
elif guess < number:
    print("Слишком маленькое")
else:
    print("Слишком большое")
Если число меньше загаданного → подсказка
Если больше → другая подсказка
8. Уменьшение попыток
attempts -= 1
print("Осталось попыток:", attempts)
attempts -= 1 — уменьшаем количество попыток на 1
Показываем, сколько осталось
9. Проигрыш
if attempts == 0:
    print("Ты проиграл. Загаданное число было:", number)

Если попытки закончились — показываем правильный ответ.

🎮 Пример
Угадай число от 1 до 100. У тебя 7 попыток.

Введи число: 30
Слишком маленькое
Осталось попыток: 6
