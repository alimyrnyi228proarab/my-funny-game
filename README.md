# my-funny-game


Это простая игра на Python. Программа загадывает число от 1 до 100, а игрок должен его угадать.

## Правила игры

У тебя есть 7 попыток.  
После каждого введённого числа программа пишет, загаданное число больше или меньше.

## Как запустить

Открой файл `coffee.py` через Python.

```bash
python coffee.py
